基于 Jetpack Compose + MVVM 架构开发的天气应用，使用 Retrofit 对接和风天气 API，实现城市搜索、实时天气与预报展示，结合 StateFlow 响应式管理状态，并通过 DataStore 持久化用户配置，所有界面元素基于 Kotlin 代码动态绘制，零图片资源依赖。
